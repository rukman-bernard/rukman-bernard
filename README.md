## 🧪 Independent System Design — NKA Academic Management System

**NKA Academic Management System** is my BCS Professional Graduate Diploma project and one of my strongest examples of structured software engineering.

It models academic administration workflows using UK-style programme structures, levels, modules, awards, semesters, and configurable academic pathways.

Key aspects:

- **Tech stack:** Laravel 11, Livewire 3, AdminLTE 3, Redis, Docker, MariaDB
- **Academic modelling:** programmes, levels, modules, awards, configs, semesters
- **Access control:** multi-guard authentication, roles, and permissions
- **System design:** structured data models, separation of concerns, caching, documentation, and maintainable Laravel architecture
- **Research direction:** connecting practical software design with higher education administration and academic framework concepts

Repository:

- **NKA Academic Management System:** [https://github.com/rukman-b/nka-academic-management-system.git](https://github.com/rukman-b/nka-academic-management-system.git)
